# 📒 Agenda de Contatos em C

Sistema simples de gerenciamento de contatos desenvolvido em linguagem C.

## 🚀 Funcionalidades

- Adicionar contato
- Listar contatos
- Buscar contato pelo nome
- Remover contato

## 🧠 Conceitos aplicados

- Struct
- Vetor de structs
- Manipulação de strings (`strcmp`)
- Menu interativo com `do-while`
- Estrutura `switch-case`

## 💻 Como compilar

Windows:

```bash
gcc agenda.c -o agenda.exe
agenda.exe
```

Linux / Mac:

```bash
gcc agenda.c -o agenda
./agenda
```
